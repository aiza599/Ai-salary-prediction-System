# 📌 Overview

AI Salary Prediction System is a Machine Learning web application that predicts an employee's expected salary based on professional experience, test score, and interview score.

The project demonstrates the complete machine learning workflow, including data preprocessing, model training, model serialization, and deployment through a Flask web application.

---

# ✨ Features

* Predict salary using Machine Learning
* Interactive web interface
* Fast real-time predictions
* Trained regression model
* Clean and responsive UI
* Easy deployment with Flask

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/aiza599/Ai-Project-.git
```

Move into the project

```bash
cd Ai-Project-
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser and visit:

```text
http://127.0.0.1:5000
```

# 📊 Machine Learning Workflow

1. Load the salary dataset
2. Clean and preprocess the data
3. Train the regression model
4. Save the trained model
5. Load the model in Flask
6. Predict salary from user input
